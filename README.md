# Card Game Hub (Python)

A terminal-based Python card game collection where you can play:

- Blackjack
- War
- Go Fish
- Five-Card Poker (showdown)
- Baccarat

## Run

```bash
python3 card_games.py
```

Follow the on-screen prompts to choose and play a game.

## Session tracking

Use menu option `6` to view your current session record (wins/losses/ties) for each game.
